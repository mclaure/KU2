## Installing

1. Download the code
2. Enter into each folder src/api-docs, src/api-kudos, src/api-stats, api-users and run:
```
    npm install
```
3. Run the following commnad to start the node server 
 
   default ports
 *  api-docs:7070
 *  api-kudos:8000
 *  api-stats:7000
 *  api-users:9000:
```
    node server.js
```
